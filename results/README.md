# Results Directory

This directory is for storing generated outputs from the Ghost Rank code.

## Expected Contents

When you run the analysis scripts, the following files will be generated here:

| File | Generated By | Description |
|------|--------------|-------------|
| `fig1_calibration_curve.png` | `generate_figures.py` | Diffusion law calibration |
| `fig2_monster_parade.png` | `generate_figures.py` | Monster curve rankings |
| `fig3_d3_anomaly.png` | `generate_figures.py` | d3 residual analysis |
| `ghosts.csv` | `ghost_detector.py` | Detected Ghost curves |
| `calibration_curve.json` | `calibration.py` | Fit parameters |

## Generating Results

```bash
# From the code/ directory:
python generate_figures.py   # Creates figures
python ghost_detector.py --input ../data/your_data.csv  # Detects ghosts
python calibration.py        # Generates calibration report
```
